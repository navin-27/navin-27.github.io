Projects

Crop Yield Prediction using Machine Learning Technique

Tech Stack: Python, Pandas, Scikit-learn, Jupyter Notebook
In this project, I developed a machine learning model to predict crop yield based on key agricultural and environmental factors. The goal was to assist farmers, agronomists, and policymakers in estimating potential yield outcomes using historical data and climate indicators.
The dataset included more than 10,000 rows of real-world crop information such as state, district, crop year, crop type, season, temperature, humidity, soil moisture, and cultivation area. I used Python and the Pandas library for data cleaning and preprocessing, including handling missing values and encoding categorical features using LabelEncoder.
For modeling, I employed a RandomForestRegressor from the Scikit-learn library with 1000 estimators. The dataset was split into training and test sets using train_test_split, and model performance was evaluated using Mean Squared Error (MSE) and R² score. The model achieved a high level of accuracy with an R² score of approximately 0.88 on the test set, indicating strong predictive capability.
To make the project interactive, I implemented a command-line interface using Python’s input() function. Users can input values like state, district, crop, season, temperature, humidity, soil moisture, and area to receive a predicted yield output from the trained model.
This project demonstrates my ability to handle end-to-end machine learning workflows—from data preprocessing and model training to evaluation and basic user interaction—all within a reproducible and interpretable Jupyter Notebook environment.

Online Bus pass management system

Tech Stack: TypeScript, CSS, JavaScript, PHP, MySQL
The Online Bus Pass System is a full-stack web application designed to simplify and digitize the process of applying for and managing bus passes. The project eliminates the need for manual paperwork and long queues by enabling students and commuters to apply, renew, and manage their bus passes entirely online.
The frontend was developed using HTML, CSS, and JavaScript, delivering a clean and user-friendly interface. On the backend, PHP was used to handle server-side logic, form processing, and communication with the database. Data storage and retrieval were managed using a MySQL database, which stores user information, application details, and pass status.
The application supports secure user authentication, allowing students to register and log in to apply for a new bus pass or renew an existing one. An integrated admin panel allows authorized personnel to review applications, verify details, and approve or reject passes. Users also receive real-time updates on the status of their applications, enhancing transparency and engagement.
This project demonstrates my capability to build a full-stack CRUD-based application, integrate database connectivity, and manage user roles and workflows. It highlights my skills in web development, backend scripting, and practical database management.
